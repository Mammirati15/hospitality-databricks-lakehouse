# Dataset

## Dataset Chosen
Hotel Booking Demand Dataset (commonly used public dataset)

## Why this dataset
- Contains booking-level rows suitable for Bronze/Silver/Gold
- Supports real hospitality metrics (Occupancy proxy, ADR, cancellations, segments)
- Great for SQL practice (grouping, filtering, time aggregation)

## Planed core columns (examples)
- hotel
- is cancelled
- arrival_date_year / month / day
- stays_in_weekend_nights + stays_in_week_nights
- adr
- adults / children / babies
- market_segment
- distribution_channel
